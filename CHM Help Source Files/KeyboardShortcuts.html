Overview

All 3D viewports in 3D File Tools share the same navigation controls. Additional keyboard shortcuts apply in the Integra XML ruler tool.

Orbit Controls Behavior

The orbit controls use damping: when you release the mouse after orbiting, the view continues to coast to a stop smoothly. The damping factor is 0.12.

General Keyboard Shortcuts

KeyAction
(No global keyboard shortcuts)Navigation is mouse-driven. Use the toolbar buttons for camera reset, zoom fit, wireframe, etc.

Ruler Tool Keyboard Shortcuts (Integra XML Only)

KeyAction

EscCancel the current ruler measurement and restore orbit controls
⌘ Cmd / Ctrl + dragOrbit the camera while the ruler is active (temporarily enables orbit)

Converter Rotation Shortcuts

In the Convert to .x panel, rotation is applied using the buttons in the bottom controls bar (X+90, X−90, Y+90, Y−90, Z+90, Z−90). There are no keyboard shortcuts for these.
