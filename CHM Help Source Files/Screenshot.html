3D File Tools » Screenshot

Overview

The screenshot feature captures the current 3D viewport as a still image. It is available in all panels (3D Viewer, Plate Placer) via the Screenshot button in the bottom controls bar.

Capturing a Screenshot

  1. Orbit and zoom the viewport to the desired camera angle
  2. Click the Screenshot button (camera icon) in the bottom controls bar
  3. The Screenshot modal opens, showing a preview of the captured image
  4. Adjust the display options (grid, background)
  5. Click a save button to download the image

Screenshot Modal

Preview

The center of the modal shows a preview of the image as it will be saved. The preview updates immediately when you toggle the grid or background options.

Display Options

ToggleDefaultDescription
Show GridOnWhen enabled, the reference grid plane is included in the screenshot. Disable to get a clean image of the model only.
Show BackgroundOnWhen enabled, the scene background color is included. Disable to get a transparent background (PNG only; JPG does not support transparency).

Save Formats

ButtonFormatNotes
Save .PNGPNG (lossless)Best quality. Supports transparency when Show Background is disabled.
Save .JPGJPEG (lossy)Smaller file size. Transparency is not supported — the background is white when Show Background is off.
Save .SVGSVG (vector)Saves the viewport canvas as a vector image. Model geometry is rasterized into the SVG; this is not a true vector export of the 3D mesh.

Closing the Modal

Click the × close button in the modal header to dismiss without saving.

Notes

  • The screenshot resolution matches the current canvas size at the device pixel ratio (×2 on high-DPI displays)
  • The filename is automatically generated from the loaded model name: e.g. mypart_screenshot.png
  • The screenshot captures exactly what is visible in the viewport, including the current light/dark theme colors
