Overview

The Integra Generator (also called the Labware Generator) creates 3D .x model files for SBS/ANSI-format microplates from parametric inputs. You can configure every aspect of the plate geometry manually, or import an existing XML labware definition file to pre-populate the parameters.

Panel Layout

The panel uses a two-column layout:

  • Left column (parameter panel) — Import button, parameter form, and action buttons
  • Right column (3D preview) — Live 3D viewport with floating toolbar; updates when you generate a model

Importing an XML Labware Definition

Click Open XML File (or drag and drop an .xml file onto the viewport) to import a Hamilton or Integra XML labware definition. The parser extracts plate dimensions, well count and geometry, and A1 position, then populates all parameter fields automatically.

After import, review the populated values and make any corrections before generating. The status display below the button shows the loaded file name or “No file loaded — edit values manually or import XML”.

Footprint (SBS/ANSI)

The SBS compliance badge next to the section title turns green (SBS Compliant) when the footprint dimensions match ANSI/SLAS 1-2004 standard values, or red (Non-standard) otherwise.

FieldDefaultDescription
Length (mm)127.76Overall plate length along the long axis
Width (mm)85.48Overall plate width along the short axis
Total Height (mm)14.35Overall plate height including lid/rim (if applicable)

Click Apply SBS Defaults to reset the footprint to the ANSI/SLAS 1-2004 standard values (127.76 × 85.48 mm).

Wells

FieldDefaultDescription
Rows8Number of well rows (A-H for a 96-well plate)
Columns12Number of well columns (1-12 for a 96-well plate)
Row Gap (mm)9.00Center-to-center distance between rows

The well count label above the fields shows the computed total (e.g. “96 wells (8×12)”). SBS standard spacing for 96-well plates is 9.0 mm; 384-well plates use 4.5 mm; 1536-well plates use 2.25 mm.

Generating and Saving

Ruler Tool

The ruler tool is available in the 3D preview via the ruler button (ruler icon) in the floating toolbar. It allows you to measure distances between two points on the generated model surface.

Using the Ruler

  1. Enable the ruler by clicking the ruler icon in the toolbar. The camera orbit controls are temporarily suspended.
  2. Click on a point on the model to set the start point
  3. Move the mouse to see a live distance readout at the bottom of the viewport
  4. Click a second point to finalize the measurement
  5. Press Esc to cancel the measurement and restore orbit controls

Axis Lock

Press X, Y, or Z while the ruler is active to lock the measurement along that axis only. Press the same key again to unlock. The readout shows the constrained distance along the locked axis.
