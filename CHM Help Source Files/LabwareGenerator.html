Overview

The Labware Generator creates 3D .x model files for SBS/ANSI-format microplates from parametric inputs. You can configure every aspect of the plate geometry manually, or import an existing labware definition file to pre-populate the parameters. Two import formats are supported:

  • Integra XML — Single .xml file with plate and well definitions
  • Hamilton — Paired .rck (rack) and .ctr (container) files from Hamilton VENUS software

Panel Layout

The panel uses a two-column layout:

  • Left column (parameter panel) — Import tabs (Integra XML / Hamilton), parameter form, and action buttons
  • Right column (3D preview) — Live 3D viewport with floating toolbar; updates when you generate a model

Importing Labware Definitions

Integra XML Import

Click Open XML File (or drag and drop an .xml file onto the viewport) to import an Integra XML labware definition. The parser extracts plate dimensions, well count and geometry, and A1 position, then populates all parameter fields automatically.

After import, review the populated values and make any corrections before generating. The status display below the button shows the loaded file name or “No file loaded — edit values manually or import”.

Hamilton Import (.rck + .ctr)

Hamilton labware definitions use two paired files:

  • .rck (rack file) — Defines the plate footprint, grid layout (rows, columns), well spacing, A1 offset (BndryX/BndryY), outer profile segments, and references the container file
  • .ctr (container file) — Defines the individual well geometry: shape (cylindrical or square), depth, diameter segments from top to bottom, and bottom shape (flat, hemisphere, V-bottom, tapered)

To import Hamilton labware:

  1. Click the Hamilton tab in the Import Definition section
  2. Click Open Rack File (.rck) and select the rack definition
  3. Click Open Container File (.ctr) and select the matching container definition
  4. Click Import Hamilton Labware to parse both files, populate the form fields, and generate the 3D model

You can also drag and drop both .rck and .ctr files simultaneously onto the viewport — if both are present, the model is generated automatically.

Footprint (SBS/ANSI)

The SBS compliance badge next to the section title turns green (SBS Compliant) when the footprint dimensions match ANSI/SLAS 1-2004 standard values, or red (Non-standard) otherwise.

Length (mm)127.76Overall plate length along the long axis

Click Apply SBS Defaults to reset the footprint to the ANSI/SLAS 1-2004 standard values (127.76 × 85.48 mm).

Ruler Tool

The ruler tool is available in the 3D preview via the ruler button (ruler icon) in the floating toolbar. It allows you to measure distances between two points on the generated model surface.

Using the Ruler

  1. Enable the ruler by clicking the ruler icon in the toolbar. The camera orbit controls are temporarily suspended.
  2. Click on a point on the model to set the start point
  3. Move the mouse to see a live distance readout at the bottom of the viewport
  4. Click a second point to finalize the measurement
  5. Press Esc to cancel the measurement and restore orbit controls

Axis Lock

Press X, Y, or Z while the ruler is active to lock the measurement along that axis only. Press the same key again to unlock. The readout shows the constrained distance along the locked axis.
