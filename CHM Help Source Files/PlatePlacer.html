Plate Placer
3D File Tools » Plate Placer

Overview

The Plate Placer tool loads a custom .x model alongside a reference 96-well microplate and allows you to visually position both objects. It calculates the difference in position as ΔX, ΔY, ΔZ offset values that can be entered directly into Hamilton VENUS labware definitions.

Use Case

When integrating a new piece of equipment (a tip box adapter, a tube rack, a custom labware carrier) into a Hamilton VENUS liquid handling system, you need to specify the labware position offset relative to a reference point on the deck. The Plate Placer tool lets you:

  1. Load the 3D model of the new equipment
  2. Visually align it against the standard 96-well plate (which serves as the reference coordinate origin)
  3. Note the calculated offset values
  4. Enter those values into the VENUS labware definition

Opening a Model

Click Open in the bottom controls bar, or drag and drop a .x or .hxx file onto the viewport to load your custom model. The 96-well plate reference model (96WellPlate.x) is always loaded automatically in the background. Both appear in the same scene.

Tip: If you already loaded a model in the 3D Viewer, switching to the Plate Placer automatically loads the same model.

Placement Offsets Panel

The Placement Offsets overlay appears in the bottom-right corner of the viewport. Click the – button to collapse it; click again to expand.

Model Position

These fields set the position of your custom model in scene coordinates (millimeters):

FieldDescription
XModel X position (mm)
YModel Y position (mm)
ZModel Z position (mm)

Plate Position & Scale

These fields control the reference plate position and overall scale:

FieldDescription
XPlate reference X position (mm)
YPlate reference Y position (mm)
ZPlate reference Z position (mm)
ScaleUniform scale factor for the plate model (1.0 = 100%). The percentage is displayed next to the field.

Calculated Offset

The Calculated Offset section shows the difference between the model position and the plate position:

FieldCalculation
ΔXModel X − Plate X
ΔYModel Y − Plate Y
ΔZModel Z − Plate Z

These values are the offsets to enter into your labware definition.

Camera Navigation

Navigation is identical to the 3D Viewer: orbit (left-click drag), pan (right-click drag), zoom (scroll wheel). Use the floating toolbar for camera reset, zoom fit, wireframe, and perspective toggle. See 3D Toolbar Reference.

Workflow Example

  1. Load your custom adapter model in the 3D Viewer first
  2. Switch to Plate Placer — your model is loaded automatically
  3. Orbit the camera to get a good view of both objects
  4. Adjust Model Position X/Y/Z to position your model visually over the plate
  5. Read the Calculated Offset ΔX, ΔY, ΔZ values
  6. Enter those values in your VENUS labware definition as the deck position offset
