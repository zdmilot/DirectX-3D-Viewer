Convert to .x
3D File Tools » Convert to .x

Overview

The Convert to .x tool imports 3D model files in OBJ, STL, or GLB/GLTF format and exports them as DirectX ASCII .x files suitable for use in Hamilton VENUS labware definitions and related tools.

Panel Layout

The converter uses a two-column layout:

  • Left column — Main 3D viewport with floating toolbar and bottom controls bar
  • Right column — Six synchronized orthographic views (Top, Front, Right, Bottom, Back, Left)

Opening a File

Click Open in the bottom controls bar or drag and drop a supported 3D file onto the main viewport. A format badge (e.g. .OBJ, .STL, .GLB) appears in the top-left corner of the viewport after loading to confirm the detected format.

Supported input formats:

ExtensionFormat
.obj / .mtlWavefront OBJ. If an .mtl file with the same base name exists in the same folder, materials are loaded automatically. You can also select both .obj and .mtl files at once using multi-select.
.stlSTereoLithography (binary or ASCII)
.glbGL Transmission Format binary package
.gltfGL Transmission Format JSON with external binary/texture refs
.xDirectX ASCII mesh (can be re-exported after reorientation)

Six Orthographic Views

The right column contains six small orthographic viewports that always show the model from fixed world-space directions. They update automatically whenever you rotate the model.

ViewCamera Direction
TopLooking down the −Y axis
FrontLooking down the −Z axis
RightLooking down the −X axis
BottomLooking down the +Y axis
BackLooking down the +Z axis
LeftLooking down the +X axis

Use these views to verify the orientation of the model before exporting, especially to confirm which side faces up and which end is the front.

Rotation Controls

The bottom controls bar contains rotation buttons to reorient the model:

ButtonAction
X+90Rotate 90° around the X axis (positive)
X−90Rotate 90° around the X axis (negative)
Y+90Rotate 90° around the Y axis (positive)
Y−90Rotate 90° around the Y axis (negative)
Z+90Rotate 90° around the Z axis (positive)
Z−90Rotate 90° around the Z axis (negative)
Reset (undo icon)Reverts all rotations to the original loaded orientation

The current cumulative rotation is displayed as X: 0° Y: 0° Z: 0° next to the rotation buttons. Rotations are cumulative: clicking X+90 twice results in a 180° rotation around X.

Exporting as .x

Once the model is correctly oriented, click Export .x in the bottom controls bar. The button is disabled until a model is loaded. The exported file is named after the original input file (e.g., importing mypart.obj exports mypart.x).

What Gets Exported

  • All mesh geometry (vertices, faces/indices) after applying the cumulative rotation
  • Per-face material assignments and diffuse colors (from OBJ MTL files or GLB materials)
  • A single Frame containing a Mesh node in DirectX .x ASCII format

See DirectX .x File Format for details on the output format.

Camera Navigation (Main Viewport)

The main viewport supports full orbit, pan, and zoom navigation. The floating toolbar provides camera controls identical to those in the 3D Viewer. See 3D Toolbar Reference and Keyboard & Mouse Controls.

Workflow Example: Converting an STL to .x

  1. Click Convert to .x in the left sidebar
  2. Click Open and select your .stl file
  3. The model appears in the main viewport and all six ortho views update
  4. Inspect the orthographic views to confirm orientation. If the model is lying on its side, click X+90 or X−90 to stand it up
  5. Confirm the orientation is correct in the six views
  6. Click Export .x — a .x file is downloaded
