3D File Tools

Overview

3D File Tools is a browser-based application for viewing, converting, and generating DirectX .x and Hamilton .hxx 3D model files. It runs entirely in the browser without any server-side installation and supports drag-and-drop file loading.

The suite is designed for laboratory automation engineers and hardware developers who work with Hamilton liquid handling systems and need to inspect or create 3D geometry for labware and instrument components.

Key Capabilities

  • 3D Viewer - Load and inspect .x (DirectX ASCII mesh) and .hxx (Hamilton 3D binary) files with a full 3D scene. Supports wireframe mode, perspective and orthographic projection, camera orbit/pan/zoom, axis gizmo, and live camera position display.
  • File Conversion - Import OBJ, STL, and GLB/GLTF files and export them as DirectX .x files. Supports cumulative rotation in 90° increments before export. Six synchronized orthographic views (Top, Front, Right, Bottom, Back, Left) update live as you rotate.
  • Plate Placer - Load a custom .x model and overlay it with a reference 96-well microplate to visually calculate X/Y/Z placement offsets. Useful for calibrating labware definitions in Hamilton VENUS.
  • Integra Generator - Build SBS-format labware 3D models from parametric inputs or imported XML labware definition files. Supports plates, well geometry, A1 positioning, and exports to .x. Includes a built-in ruler tool for measuring distances in the 3D scene.
  • Export - Export any loaded model to OBJ, STL, GLB, or HXX format directly from the browser.
  • Screenshot - Capture the 3D viewport as PNG, JPG, or SVG with options to include or exclude the grid and background.
  • Transform - Rotate the loaded model ±90° around X/Y/Z axes and mirror across any axis. Transforms are applied to the underlying .x file data and can be saved back to a .x file.
  • Dark / Light Theme - Toggle between light and dark rendering modes. All viewports adapt simultaneously.
  • HXX Support - Read Hamilton binary .hxx archives containing gzip-compressed .x data and optional PNG texture sections. Transparent to the user: dropped .hxx files open just like .x files.

System Requirements

  • Browser: Any modern browser with WebGL support (Chrome, Firefox, Edge, Safari)
  • Operating System: Any OS (runs entirely in the browser)
  • Server: No server required — open index.html directly, or serve with any static HTTP server
  • External Dependencies: None (all libraries bundled in js/lib/)

Overview

Venus Library Manager is a desktop application and command-line tool for managing Hamilton VENUS libraries and related assets. Built on NW.js and Node.js, it provides a complete package lifecycle for .hxlibpkg (single library package) and .hxlibarch (multi-library archive) files.

Venus Library Manager enables laboratory automation engineers to create, import, export, update, roll back, and verify Hamilton VENUS HSL libraries through both a graphical user interface and a scriptable CLI. It replaces the error-prone manual process of copying library files into the VENUS directory tree with a structured, versioned, integrity-checked workflow.

Key Capabilities

  • Visual Library Management - Browse installed libraries as cards with icons, metadata, version, author, tags, and integrity status indicators
  • Package Import - Import single .hxlibpkg packages or multi-library .hxlibarch archives with pre-install previews and validation
  • Package Export - Export installed libraries back to portable package files, capturing their current on-disk state (mutable export). Optionally export a library with all of its recursively-resolved dependencies as a single archive
  • Package Creation - Build .hxlibpkg files from raw HSL source files and metadata using the GUI Library Packager or the create-package CLI command
  • Library Deletion - Safely remove libraries with COM-aware workflow, soft-delete history, and optional hard-delete
  • Version Rollback - Automatically cache every imported package and roll back to any previously installed version on demand
  • File Integrity Verification - SHA-256 hashing for user libraries and Hamilton metadata footer verification for system libraries
  • Library Grouping - Organize libraries into custom groups with drag-and-drop reordering and assignment
  • Hamilton Default Group - Protected, non-deletable group for Hamilton-authored libraries with password-gated authorship control
  • Visual Identity - Composited package icons with brand overlay and distinctive purple archive icons for multi-library bundles
  • VENUS Tool Integration - Launch Hamilton editors (Method, Liquid Class, Labware, HSL, System Config) directly from the application
  • Command-Line Interface - Full-featured CLI (cli.js) for scripted automation, CI/CD integration, and headless operation

System Requirements

  • Operating System: Microsoft Windows 10 or Windows 11
  • VENUS Version: 4.7 and above (recommended: 6.x)
  • Runtime: Node.js (for CLI usage independent of the GUI)
  • Default Library Path: C:\Program Files (x86)\HAMILTON\Library
  • Default Methods Path: C:\Program Files (x86)\HAMILTON\Methods
